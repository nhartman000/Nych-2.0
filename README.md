# Nych Protocol v2.0
Nych Protocol v2.0 is a symbolic / latent-space machine language stack and protocol designed for efficient multi-LLM coordination, holographic projection of state, and emoji-based compression of operations.

## Repository Status

- **Version:** v2.0 (spec-first release)
- **Code:** Python implementation not yet committed – this repo currently hosts the protocol, lexicon, rules, and operation specifications.
- **Focus:** Preparing a stable, reference-grade spec for future engines and 3-LLM holographic projection experiments.

## Structure

- `LICENSE` – Custom Nych v2.0 license (see section “License” below).
- `docs/`
  - `nych_protocol_v2.0.md` – Core protocol specification.
  - `nych_lexicon_v2.0.md` – Lexicon and symbolic definitions.
  - `nych_rules_and_conventions.md` – Usage rules, constraints, and operational semantics.
  - `nych_operation_3LLM_holographic_projection.md` – 3-LLM holographic projection instructions, design, and code notes.
- `src/` – Reserved for upcoming Python implementation.
